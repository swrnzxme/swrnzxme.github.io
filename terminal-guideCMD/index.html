Гайд по командной строке Windows

Изучите команды PowerShell и CMD с примерами и заданиями

Введение

Командная строка Windows — это мощный инструмент для управления системой, файлами, процессами и пользователями. В этом гайде вы узнаете, как использовать команды в PowerShell и CMD.

Чтобы открыть командную строку:

  • Нажмите Win + R, введите cmd или powershell.
  • Или найдите "Командная строка" или "PowerShell" в меню "Пуск".

Работа с файлами

Команды CMD

dir — показать содержимое текущей папки.
copy file.txt new_folder\ — скопировать файл.
del file.txt — удалить файл.
mkdir folder — создать папку.

Команды PowerShell

Get-ChildItem — показать содержимое папки.
Copy-Item file.txt -Destination new_folder\ — скопировать файл.
Remove-Item file.txt — удалить файл.
New-Item -Name "folder" -ItemType Directory — создать папку.

Пример

Создайте файл и переименуйте его:

echo "Hello" > draft.txt
Rename-Item draft.txt final.txt

Управление пользователями

Команды CMD

net user — список пользователей.
net user John /add — добавить пользователя.
net localgroup Administrators John /add — дать права администратора.

Команды PowerShell

New-LocalUser -Name "Anna" — создать пользователя.
Add-LocalGroupMember -Group "Administrators" -Member "Anna" — добавить в группу.

Задание

Создайте пользователя TestUser и добавьте его в группу Users.

Мониторинг процессов

Команды CMD

tasklist — список процессов.
taskkill /IM notepad.exe /F — завершить процесс.

Команды PowerShell

Get-Process — список процессов.
Stop-Process -Name "notepad" — завершить процесс.

Пример

Найдите процессы с нагрузкой CPU >100:

Get-Process | Where-Object { $_.CPU -gt 100 }
